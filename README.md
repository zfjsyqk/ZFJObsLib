## 软件功能
### 混淆相关：
#### 1、代码方面：属性混淆、类名混淆、函数混淆、方法混淆、垃圾代码、加密字符串、删除注释、关键词混淆、Xib/Storyboard控件id混淆
#### 2、资源方面：修改资源名、修改资源HASH值、图片混淆（指定16进制改色、颜色替换、全部改色）

#### 4、其他混淆：敏感词过滤、控件颜色魔改、字体颜色魔改
